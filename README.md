# 🌈 PRISM OS v3.0

> **Arch Linux** with insane glass UI

## 🚀 One-Click Build

1. **Fork this repo**
2. **Click "Actions" tab**
3. **Click "Build PRISM OS"**
4. **Click "Run workflow"**
5. **Wait 10-15 minutes**
6. **Download your ISO!**

## ✨ Features

- 🎨 i3 + Polybar + glass effects
- 🎮 Discord, Spotify, Firefox pre-installed
- 🌊 Animated GLSL wallpapers
- 🎵 Music visualizer
- 🌐 MediaTek + all network drivers
- 💎 Seamless boot - zero setup needed

## 🔥 Quick Install

```bash
# Flash to USB
dd if=prism-os-v3.0.iso of=/dev/sdX bs=4M status=progress

# Or use Ventoy - just copy ISO to USB!
```
