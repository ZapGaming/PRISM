Welcome to PRISM OS - Glass Linux Distribution

QUICK START
═══════════
1. Login: prism / prism
2. Super+Enter: Open terminal
3. Super+D: Open launcher
4. Super+Shift+Q: Close window

DESKTOP
═══════
• i3-gaps - Tiling window manager
• Polybar - Glass status bar
• Picom - Blur compositor
• Rofi - App launcher
• Kitty - GPU terminal

CONTROLS
═══════════
• Super+Arrows: Navigate windows
• Super+Shift+Arrows: Move windows
• Super+1-9: Switch workspace
• Super+Shift+1-9: Move to workspace
• Super+Shift+E: Exit
• Super+R: Resize mode

SUPPORT
═══════
• Live USB to real SSD install
• VMWare/VirtualBox compatible
• EFI and BIOS boot
